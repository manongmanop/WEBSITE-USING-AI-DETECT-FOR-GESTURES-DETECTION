==================================================
คู่มือการติดตั้งโปรแกรมโครงงาน (Installation Guide)
==================================================

โครงสร้างโฟลเดอร์:
--------------------------------------------------
1. โฟลเดอร์ "Dev Tools"
   - ใช้สำหรับเก็บไฟล์ติดตั้งโปรแกรมพื้นฐานที่จำเป็นในการรันโครงงาน
   - แนะนำให้ดาวน์โหลดและนำมาใส่ไว้ในโฟลเดอร์นี้:
     1. Node.js (เวอร์ชัน LTS) - สำหรับรันเซิร์ฟเวอร์และโปรเจกต์
     2. MongoDB Community Server - สำหรับฐานข้อมูล
     3. MongoDB Compass - สำหรับจัดการฐานข้อมูลแบบหน้าต่าง UI
     4. VS Code (Visual Studio Code) - สำหรับเปิดและแก้ไขโค้ด

2. โฟลเดอร์ "Setup"
   - ใช้สำหรับเก็บไฟล์ Source Code ของโครงงาน (ไฟล์โปรเจกต์ทั้งหมด)
   - แนะนำให้ Zip ไฟล์โปรเจกต์ (ยกเว้นโฟลเดอร์ node_modules) แล้วนำมาวางไว้ที่นี่

ขั้นตอนการติดตั้งและการตั้งค่า (Setup & Run):
--------------------------------------------------
ขั้นตอนที่ 1: การติดตั้งโปรแกรมพื้นฐาน (จากโฟลเดอร์ Dev Tools)
1. ติดตั้ง Node.js (กด Next ตามขั้นตอนมาตรฐาน)
2. ติดตั้ง MongoDB Community Server และ MongoDB Compass ให้เรียบร้อย
3. ติดตั้ง VS Code สำหรับเปิดโปรเจกต์

ขั้นตอนที่ 2: การนำ Source Code ไปใช้งาน (จากโฟลเดอร์ Setup)
1. นำไฟล์โครงงาน (Source Code) จากโฟลเดอร์ Setup ไปแตกไฟล์ (Extract) ไว้ในไดรฟ์ที่ต้องการ เช่น D:\Project
2. เปิดโปรแกรม VS Code
3. เลือก File -> Open Folder... แล้วเลือกโฟลเดอร์โครงงานที่คุณเพิ่งแตกไฟล์ออกมา

ขั้นตอนที่ 3: การติดตั้งไลบรารี (Dependencies)
1. ใน VS Code เปิด Terminal โดยไปที่เมนู Terminal -> New Terminal
2. พิมพ์คำสั่งต่อไปนี้แล้วกด Enter เพื่อติดตั้งแพ็กเกจที่จำเป็น:
   npm install
   (รอจนกว่าการติดตั้งจะเสร็จสมบูรณ์ จะมีโฟลเดอร์ node_modules ปรากฏขึ้น)

ขั้นตอนที่ 4: การรันระบบฐานข้อมูล (MongoDB)
1. ตรวจสอบให้แน่ใจว่า MongoDB Service กำลังทำงานอยู่ในเครื่อง
2. สามารถเปิด MongoDB Compass แล้วกด Connect ไปที่พอร์ตเริ่มต้น (mongodb://localhost:27017) เพื่อเช็คการเชื่อมต่อ

ขั้นตอนที่ 5: การรันโครงงาน (Start Project)
1. ใน Terminal ของ VS Code พิมพ์คำสั่งต่อไปนี้แล้วกด Enter:
   npm run dev
2. ระบบจะทำการเปิดทั้งฝั่ง Backend (Server) และ Frontend (Client) ขึ้นมา
3. สังเกตใน Terminal จะมี URL ขึ้นมา (เช่น http://localhost:5173 หรือ 3000)
4. เปิด Web Browser (แนะนำ Google Chrome) และพิมพ์ URL ดังกล่าวเพื่อเข้าใช้งานแอปพลิเคชัน

ขั้นตอนที่ 6: การจำลองข้อมูลตั้งต้น (Seeding Data) - *เฉพาะครั้งแรก*
1. หากฐานข้อมูลยังว่างเปล่า ให้เปิด Terminal อันใหม่
2. รันคำสั่งสร้างข้อมูลจำลอง (ถ้ามี) ตัวอย่างเช่น:
   node test-ai-seed.cjs
3. ข้อมูลเริ่มต้น (เช่น โปรแกรมออกกำลังกาย) จะถูกเพิ่มเข้าไปในระบบ

==================================================
หากพบปัญหา:
- ให้ตรวจสอบว่า MongoDB รันอยู่หรือไม่
- ตรวจสอบเวอร์ชันของ Node.js ว่าเป็นเวอร์ชันที่รองรับหรือไม่
- หาก npm run dev ไม่ทำงาน ให้ลองลบโฟลเดอร์ node_modules และไฟล์ package-lock.json แล้วรัน npm install ใหม่อีกครั้ง
==================================================
